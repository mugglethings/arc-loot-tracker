# Arc Loot Tracker

A GUI-based tool for tracking and cataloging loot from game locations. Captures inventory grids via screenshot, recognizes items using template matching, and logs loot data with map, location, and container information.

## Features

- **Screenshot Capture**: Press F8 to scan inventory grids
- **Template Matching**: Automatically recognizes items using image comparison
- **Map & Location Tracking**: Organize loot by map, location, and container
- **Item Templates**: Builds a library of item images for recognition
- **Loot Logging**: Records all loot to a centralized loot.txt file

## Usage

Run the application and select:
1. Game map
2. Condition (e.g., Day, Night)
3. Location within the map
4. Container type
5. Number of grid cells to scan (1-12)
6. (Eventually Item name)
7. Quantity

Press F8 when ready to capture the inventory grid.
